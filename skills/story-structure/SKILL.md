# Story Structure (the circle + the spine)

Two compatible tools from the writing world that both encode **causation over sequence** — the same rule as `therefore-but-not-and-then`, and (notably) both born from *improv*.

## The Story Spine (Kenn Adams, 1991 — an improv exercise)
> Once upon a time… / Every day… / **But, one day**… / **Because of that**… / **Because of that**… / **Because of that**… / Until finally… / And, ever since then…

The repeated *"because of that"* is the engine: each consequence triggers the next, raising stakes through "interconnected events rather than random incidents" ([Aerogramme Studio](https://www.aerogrammestudio.com/2013/03/22/the-story-spine-pixars-4th-rule-of-storytelling/)). Born in improv (Freestyle Repertory Theater), reached Pixar later — which is why it fits a live, improvised table so well.

## The Story Circle (Dan Harmon, an 8-step Hero's-Journey distillation)
**Comfort → Want → Go → Search → Find → Take → Return → Change** ([Reedsy](https://reedsy.com/blog/guide/story-structure/dan-harmon-story-circle/)). It fuses *external plot* (go, search, find, take) with *internal change* (the character returns to where they started, psychologically altered). The causation rule: "each step triggers the next through consequence, not coincidence."

## How to use it at the table / in chat
- **One session = one circle.** The party leaves comfort, wants something, crosses a threshold, pays a price, and returns to the tavern *changed*. End each session having turned the wheel once.
- **The campaign = circles within circles.** Small arcs nest inside the big one; the protagonists evolve across episodes.
- **Run the spine as a planning check:** write your beats, and if "and then" fits between two of them, you have a list, not a story — find the "because of that." (This is the same test as `pacing-and-spotlight` and `therefore-but-not-and-then`.)
- **Always end on "until finally / but one day"** — the hook that demands the next beat.

> Convergence worth noting: the D&D improv spine (`yes-and-improv`), Pixar's spine, and Harmon's circle are *the same principle* — causal momentum — discovered in three places. The same way the presiding covenant turned up inside Mercer's DM craft.
